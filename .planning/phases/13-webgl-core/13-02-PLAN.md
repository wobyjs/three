<objective>
Fix all systemic broken patterns in the 256 demo/src/*.tsx files so every registered demo can
render without import errors. Then run an automated Playwright sweep to classify which demos pass
and which need individual repair.

Purpose: Two known patterns break the majority of demos at runtime: (1) `<OrbitControls>` uses the
Three.js class constructor as if it were a JSX component — JSX treats uppercase tags as React
components and calls them as functions, which crashes because OrbitControls is a class, not a
Woby component; (2) `<scene>` elements whose module-registration side-effect import is missing
cause "scene is not a constructor" runtime errors. Fixing these two patterns en masse unblocks
~90% of demos before any individual repair is needed.

Output:
- All demo/src/*.tsx files with corrected imports and JSX tags
- playwright.test/test-results/demo-report.json listing pass/fail per demo ID
- Screenshots under playwright.test/test-results/screenshots/
</objective>

How JSX element names resolve to Three.js constructors
  JSX tag -> customElement2name(tag) -> toUpper(name) -> Three[name] constructor

toUpper key transformations (from code/lib/utils.ts line 8):
  'webglRenderer' -> s.replace('gl','GL') -> 'WebGLRenderer' -> Three['WebGLRenderer'] ✓
  'webGLRenderer' -> no 'gl' substring (already GL) -> 'WebGLRenderer' ✓  (BOTH work)
  'orbitControls' -> no match -> 'OrbitControls' -> Three['OrbitControls'] ✓
  'OrbitControls' -> uppercase tag = treated as Woby function component -> CRASHES ✗

Module registration pattern (each side-effect import populates window.__$$Three$$__):
  import '@woby/three/src/scenes/Scene'
    -> sets Three.Scene = Scene, registers consParams.scene, objProps.scene
    -> enables <scene> JSX element

  import '@woby/three/examples/jsm/controls/OrbitControls'
    -> sets Three.OrbitControls = TOrbitControls, registers consParams.orbitControls
    -> enables <orbitControls> JSX element

    const DEMO_DIR = new URL('./src', import.meta.url).pathname
    const files = readdirSync(DEMO_DIR).filter(f => f.endsWith('.tsx'))
    let changed = 0

    for (const file of files) {
      const fpath = join(DEMO_DIR, file)
      let src = readFileSync(fpath, 'utf8')
      if (!src.includes('<OrbitControls')) continue

      // Step A: ensure side-effect import exists
      const sideEffect = "import '@woby/three/examples/jsm/controls/OrbitControls'"
      if (!src.includes(sideEffect)) {
        // Insert after last '@woby/three/src/' import or after last '@woby/three/examples/jsm/' import
        const insertAfter = src.match(/import '.*@woby\/three\/(src|examples\/jsm)\/[^']+'\n/g)
        if (insertAfter) {
          const lastImport = insertAfter[insertAfter.length - 1]
          const pos = src.lastIndexOf(lastImport) + lastImport.length
          src = src.slice(0, pos) + sideEffect + '\n' + src.slice(pos)

          // Prepend after the last import statement
          src = src.replace(/^(import[^;]+;\n)(?!import)/m, '$1' + sideEffect + '\n')
        }
      }

      // Step B: replace named import with side-effect (if OrbitControls only used as JSX)
      // Check for non-JSX usages
      const nonJsx = src.match(/(?<!<\/?)(: OrbitControls|new OrbitControls|OrbitControls\.)/g)
      if (!nonJsx) {

        src = src.replace(/import \{ OrbitControls \} from '@woby\/three\/examples\/jsm\/controls\/OrbitControls'\n/g, '')

      src = src.replace(/<OrbitControls(\s)/g, '<orbitControls$1')
      src = src.replace(/<OrbitControls\/>/g, '<orbitControls/>')
      src = src.replace(/<OrbitControls>/g, '<orbitControls>')

    Implementation: Write a Node.js script at demo/fix-scene-imports.mjs and run it.

    Script content:
    ```javascript
    import { readFileSync, writeFileSync, readdirSync } from 'fs'
    import { join } from 'path'

    const DEMO_DIR = new URL('./src', import.meta.url).pathname
    const files = readdirSync(DEMO_DIR).filter(f => f.endsWith('.tsx'))
    let changed = 0

    const SCENE_IMPORT = "import '@woby/three/src/scenes/Scene'"

    for (const file of files) {
      const fpath = join(DEMO_DIR, file)
      let src = readFileSync(fpath, 'utf8')

      // Skip if already has Scene import
      if (src.includes(SCENE_IMPORT) || src.includes('scenes/Scene')) continue
      // Skip if doesn't use <scene
      if (!src.includes('<scene')) continue

      // Find insertion point: after the last '@woby/three/src/' side-effect import
      // Strategy: find the last line matching /^import '@woby\/three\/src\//
      const lines = src.split('\n')
      let lastSrcImportIdx = -1
      for (let i = 0; i < lines.length; i++) {
        if (lines[i].match(/^import '.*@woby\/three\/src\//)) {
          lastSrcImportIdx = i
        }
      }

      if (lastSrcImportIdx >= 0) {
        lines.splice(lastSrcImportIdx + 1, 0, SCENE_IMPORT)
      } else {
        // No src imports found — insert after the last import line
        let lastImportIdx = -1
        for (let i = 0; i < lines.length; i++) {
          if (lines[i].startsWith('import ')) lastImportIdx = i
        }
        if (lastImportIdx >= 0) {
          lines.splice(lastImportIdx + 1, 0, SCENE_IMPORT)
        } else {
          lines.unshift(SCENE_IMPORT)
        }
      }

      writeFileSync(fpath, lines.join('\n'))
      changed++
    }
    console.log(`Added Scene import to ${changed} files`)
    ```
